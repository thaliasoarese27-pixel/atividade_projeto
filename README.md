# Atividade Prática

O repositório foi criado para atender ao enunciado da atividade prática de Gerenciamento de Configuração.    

## Objetivo
- Demonstrar o uso do Git e GitHub para controle de versão.  
- Criar e modificar artefatos locais e atualizar no repositório remoto.  
- Evidenciar o processo de colaboração em um projeto de software.  

## Artefatos
- `documentacao.txt` → requisitos fictícios do projeto.  
- `codigo_exemplo.js` → código simples em JavaScript.  
- `README.md` → descrição do projeto.
